# Inventory Management System

A full-stack Inventory Management System built with FastAPI, Streamlit, and MySQL.

## 🚀 Tech Stack
- FastAPI (Backend API)
- Streamlit (Frontend UI)
- MySQL (Railway Database)
- SQLAlchemy (ORM)
- Render (Backend Deployment)

## 🌐 Live Links
- 🔗 Test: https://inventory-management-5pztrn2fhdcmfmdpnurg8j.streamlit.app/

## ⚙️ Features
- Add, update, delete products
- View inventory list
- REST API backend
- Cloud database integration

## 📌 Architecture
Streamlit → FastAPI → MySQL
